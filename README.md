# Chinese Historical Taxation Database

A comprehensive academic database of taxation systems in imperial China with comparative analysis of contemporary civilizations.

## Overview

This repository contains systematically collected data on taxation rates, collection methods, and fiscal burdens across major Chinese dynasties, with comparative data from contemporary civilizations including the Roman Empire, Byzantine Empire, and medieval European states.

## Project Goals

- Provide reliable, academically-sourced data on Chinese imperial taxation
- Enable comparative analysis with other ancient/medieval fiscal systems
- Support research in economic history, comparative politics, and historical sociology
- Establish baseline data for understanding fiscal state capacity in pre-modern China

## Key Features

- **Academic rigor**: All data sourced from peer-reviewed academic publications
- **Multi-language sources**: Research conducted in Chinese, English, Japanese, and other languages
- **Structured format**: JSON schema for easy data analysis and visualization
- **Version controlled**: Complete change history and data provenance
- **Open access**: Available under Creative Commons license

## Data Coverage

### Chinese Dynasties
- Han Dynasty (206 BCE - 220 CE)
- Tang Dynasty (618 - 907 CE) 
- Song Dynasty (960 - 1279 CE)
- Ming Dynasty (1368 - 1644 CE)
- Qing Dynasty (1644 - 1912 CE)

### Tax Categories
- Land taxes (田赋)
- Poll taxes (人头税)
- Commercial taxes (商税)
- Salt and iron monopolies (盐铁专卖)
- Corvée labor (徭役)

### Comparative Civilizations
- Roman Empire
- Byzantine Empire
- Abbasid Caliphate
- Medieval European kingdoms
- Tokugawa Japan
- Joseon Korea

## Usage

### Data Access
All data files are available in JSON format in the `data/` directory. Each file includes:
- Tax rates and collection methods
- Historical context and time periods
- Source attribution and reliability ratings
- Comparative benchmarks
